## Getting Started with GitHub Codespaces

This repository is configured with a development container that includes PostgreSQL and essential tools.

### Starting a Codespace
1. Click the green "Code" button
2. Select "Codespaces" tab
3. Click "Create codespace on main"
4. Wait for the environment to initialize

### Accessing PostgreSQL
Once the Codespace loads, PostgreSQL is automatically running:
- **Host:** localhost
- **Port:** 5432
- **Username:** postgres
- **Password:** postgres

### Recommended VS Code Extensions
- PostgreSQL Client (cweijan.vscode-postgresql-client2)
- SQLTools (mtxr.sqltools)
- SQLTools PostgreSQL Driver (mtxr.sqltools-driver-pg)
